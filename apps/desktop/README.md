# Desktop App

Desktop direction: `Tauri + React`.

The MVP will call the local FastAPI service over `http://127.0.0.1:8765`. This folder is intentionally light until the API and first data loop are stable enough to drive screens.

First screens:

- Workbench: today's actions and risk ranking.
- Holdings: portfolio table and signal status.
- Watchlist: target stocks and trigger status.
- Stock detail: plan, data summary, signal explanation and history.
